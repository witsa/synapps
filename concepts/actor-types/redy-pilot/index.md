# REDY PILOT

Studio **1.6.0-beta**
{: .label .label-yellow }
Runtime **2.8.0**
{: .label .label-green }
REDY Pilot **1.0.0**
{: .label .label-yellow }

La suite d'acteurs REDY Pilot permet de créer des applications métiers connectées spécifique à un REDY Pilot.

{: .warning }
> Ces acteurs sont en version `BETA`{: .label .label-yellow } et peuvent évoluer dans les prochaines versions. Ils sont destinés à être utilisés avec REDY Pilot mais peuvent aussi être utilisés dans d'autres contextes.
