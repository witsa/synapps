# REDY PILOT `BETA`{: .label }

La suite d'acteurs REDY PILOT permet de créer des applications métiers connectées spécifique à un REDY PILOT.

{: .warning }
> Ces acteurs sont en version BETA et peuvent évoluer dans les prochaines versions. Ils sont destinés à être utilisés avec REDY PILOT mais peuvent aussi être utilisés dans d'autres contextes.
