# API JavaScript

La référence de l'API JavaScript de Synapps Runtime est disponible ici. Elle décrit les différentes classes, méthodes, propriétés et événements disponibles pour interagir avec les acteurs, les scène et composites, ainsi que les services qui accèdent au REDY.

{: .important-title }
> 💎 Documentation Avancée
>
> Cette documentation est destinée aux développeurs souhaitant étendre les fonctionnalités de Synapps Runtime via des scripts JavaScript. Elle nécessite une bonne compréhension de la programmation JavaScript et des concepts de Synapps.
