createBasicBinding

(
  • targetValuePath
  • setup
  • [saveInSetup=true]
)
Binding

Création d'une liaison basique.

La source est prise dans le contexte de donnée.

Parameters:

  • targetValuePath String

    Chemin de la cible

  • setup Object
  • [saveInSetup=true] Boolean optional

Returns:

Binding:

la liaison créée.

Example:

Le setup est un objet

actor.createBasicBinding('properties.width, { path: 'widthOfPanel' });

La propriété largeur de l'acteur est maintenant liée au champ widthOfPanel de son contexte de donnée.

registerInDefinitions

(
  • definition
)
private static

Enregistre la définition de l'acteur dans le dictionnaire des acteurs.

Parameters:

  • definition Object
    • type String
      • La clé unique du type de l'acteur (ex: "input/button");
    • componentName String
      • Nom complet du composant EmberJS (ex: "synapps-ui/actors/input/button").
    • prototypeSetup String
      • Object qui contient les définitions des propriétés de l'acteur.
    • [platform=native] String optional
      • Nom de la plateforme à laquelle l'acteur appartient.
