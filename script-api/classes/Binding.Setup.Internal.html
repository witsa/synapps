Binding.Setup.Internal Class

Extends Binding.Setup.Basic
Module: Synapps-UI

Setup de liaison interne.

Index

Show:

Attributes

  • canWrite
  • path
  • readMode
  • sourceType

Attributes

canWrite

Boolean optional

Inherited from Binding.Setup.Basic:

Permettre l'écriture dans la cible

Default: false

path

String required

Inherited from Binding.Setup.Basic:

Chemin pour trouver la valeur sur la source.

Example:

properties.width ou additionals.prop1

readMode

String optional

Inherited from Binding.Setup.Basic:

Mode de lecture de la valeur cible :

Default: always

Example:

  • always pour chaque changement
  • onetime pour qu'une seule lecture.

sourceType

String required

Type de source Acteur.

Example:

  • actor pour créer une liaison vers un acteur : Binding.Setup.Actor
  • relative pour créer une liaison relative : Binding.Setup.Relative
  • stage pour créer une liaison vers la scène courante ou le composite parent : Binding.Setup.Stage
