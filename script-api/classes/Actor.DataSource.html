Actor.DataSource Class

Extends Actor.BaseActor
Module: Synapps-UI

Classe de base pour fournisseur de donnée. Permet de paramétrer les requêtes et d'exposer une donnée pour des liaisons ou accessible par script.

properties.autoRefresh Boolean

Auto rafraîchissement activé?

Default: true

properties.autoRefreshDelay Number

Délai en seconde de rafraîchissement.

Default: 30

properties.classNames String

Inherited from Actor.BaseActor:

Nom de class CSS séparées par une virgule à ajouter à l'acteur. Modificateur : Rendu.

Default: ''

properties.height String

Inherited from Actor.BaseActor:

Hauteur de l'acteur Modificateur : Gabarit.

Default: 'auto'

properties.maxHeight String

Inherited from Actor.BaseActor:

Hauteur maximale de l'acteur Modificateur : Gabarit.

Default: 'auto'

properties.minHeight String

Inherited from Actor.BaseActor:

Hauteur minimale de l'acteur Modificateur : Gabarit.

Default: 'auto'

properties.mode String

Mode de requête de la donnée.

Default: 'auto'

Example:

  • auto : Récupération de la donnée à l'initialisation et possibilité de rafraîchir automatiquement,
  • manual: Les requêtes doivent être commandées par script avec la méthode request ou requestDebounce.

Inherited from Actor.BaseActor:

Default: 'inherit'

Écriture dès changement de la donnée activée?

createBasicBinding
(
  • targetValuePath
  • setup
  • [saveInSetup=true]
)
Binding

Inherited from Actor.BaseActor:

Création d'une liaison basique.

La source est prise dans le contexte de donnée.

Parameters:

  • targetValuePath String

    Chemin de la cible

  • setup Object
  • [saveInSetup=true] Boolean optional

Returns:

Binding:

la liaison créée.

Example:

Le setup est un objet

actor.createBasicBinding('properties.width, { path: 'widthOfPanel' });

La propriété largeur de l'acteur est maintenant liée au champ widthOfPanel de son contexte de donnée.
